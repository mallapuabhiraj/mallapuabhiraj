## 🚀 Projects

### 🛡️ Credit Card Fraud Detection System
> *Python · CatBoost · XGBoost · LightGBM · Optuna · Pandas · Scikit-learn*

A **production-grade ML pipeline** on 1M+ real-world transactions — with only 0.6% fraud rate. The ultimate imbalanced classification challenge, solved with rigor.

- 📐 Engineered **40+ domain-driven fraud risk signals** — geospatial anomaly detection, temporal behavioral aggregation, and transaction pattern features
- 🔒 Enforced **leakage-safe chronological validation** to faithfully simulate real-world deployment conditions
- ⚙️ Applied **Bayesian hyperparameter optimization (Optuna)** with cost-sensitive threshold calibration (FP=1, FN=6)
- 📊 Delivered **0.928 PR-AUC** and **0.998 ROC-AUC** on 555K unseen test records
- 🎯 Achieved **89% Recall @ 80% Precision** — because missing a fraudster is far costlier than a false alarm

### ❤️ Heart Disease Prediction
> *Python · Logistic Regression · Random Forest · Scikit-learn · Seaborn*

A rigorous clinical classification pipeline built with explainability and generalization at its core — where model transparency is as important as performance.

- 🔍 End-to-end pipeline: EDA → preprocessing → stratified splits → leakage-safe feature scaling
- 🔧 Tuned via **RandomizedSearchCV** with Precision-Recall based threshold optimization
- 📈 Lifted F1-score from **0.82 → 0.87** | ROC-AUC: **0.91** | Accuracy: **86.9%**
- 🧬 Delivered explainability via **odds ratios and feature importance** — essential for high-stakes healthcare decisions

### 🔥 Gas Leakage Detection & Protection System
> *Arduino C++ · Arduino Uno · MQ-135 Sensor · Buzzer · Servo Motor · LED*

A real-time embedded safety system that detects hazardous gas leaks and autonomously triggers protective responses — modeled after industrial substation safety standards.

- 🧪 Integrated **MQ-135 gas sensor** with precision threshold calibration for reliable, low-false-alarm hazard detection
- ⚡ Achieved **sub-2-second response time** within 100cm — simultaneously triggering buzzer alerts, servo-driven gas valve closure, and LED status indication
- 🏭 Architected to mirror **real-world substation safety protocols**, demonstrating production-aware embedded systems thinking
- 🔧 Showcases core embedded competencies: **real-time interrupt handling, sensor fusion, and actuator control**
